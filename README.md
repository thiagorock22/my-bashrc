# My bashrc

Some commands

```
$ extract some-file.tar # will execute tar xf some-file.tar
$ extract some-file.zip # will execute unzip some-file.zip

$ cbssh some-file.zip # will copy ~/.ssh/id_rsa.pub to clipboard

$ mcd folder1 # will create folder1 and cd folder1

$ genpasswd # will generate a hash
```
